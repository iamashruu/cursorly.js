What is Cursorly.js?

Custom CURSOR library with optional effects,Show magic with mouse / cursor.

Cursorly.js is a lightweight JavaScript library that replaces the default browser cursor with customizable icons and interactive particle effects like trail or sparkle. It’s designed to be developer-friendly, flexible, and fun — perfect for creative websites, portfolios, or experimental UI design.

Features

  • Custom Cursor Icons
  • Particle effects
  • Enable/disable cursor & particles dynamically
  • Lightweight, high-performance, canvas-based
  • Developer-friendly

Code Snippets

// Switch cursor icon dynamically
cursor.setIcon(1);

// Change particle effect dynamically
cursor.setEffect({
    name: "sparkle",
    color: "#ff4081",
    density: 10,
    size: [3, 12]
});

// Disable or enable cursor
cursor.disable();
cursor.enable();
