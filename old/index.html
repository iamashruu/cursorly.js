What is Cursorly.js?

Custom CURSOR library with optional effects,Show magic with mouse / cursor.

Cursorly.js is a lightweight JavaScript library that replaces the default browser cursor with customizable icons and interactive particle effects like trail or sparkle. It’s designed to be developer-friendly, flexible, and fun — perfect for creative websites, portfolios, or experimental UI design.

Features

  • Custom Cursor Icons
  • Particle effects
  • Enable/disable cursor & particles dynamically
  • Lightweight, high-performance, canvas-based
  • Developer-friendly

Installation

Include the library via CDN:

<script src="https://cdn.jsdelivr.net/gh/iamashruu/cursorly.js/script.js"></script>

Initialization

Initialize the cursor with default options:

const cursor = Cursorly.init({
    cursor: 0, // Index of the cursor icon (default: 0)
    effect: { name: "trail", color: "rainbow" } // Effect name and color are required
});
Note: Make sure to include the script tag before using the library in your JavaScript code.

Example

Here’s a complete example:


        <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Cursorly.js Example</title>
                <script src="https://cdn.jsdelivr.net/gh/iamashruu/cursorly.js/script.js"></script>
            </head>
            <body>
                <script>
                    const cursor = Cursorly.init({
                        cursor: 0,
                        effect: { name: "trail", color: "rainbow" }
                    });
                </script>
            </body>
        </html>

Code Snippets


//Initialize
const cursor = Cursorly.init({
    cursor: 0,
    effect: { name: "trail", color: "rainbow" }
});

// alternate with default values
const cursor = Cursorly.init();

// Switch cursor icon dynamically
cursor.setIcon(1);

// Change particle effect dynamically
cursor.setEffect({
    name: "sparkle",
    color: "#ff4081",
    density: 10,
    size: [3, 12]
});

//Disable or enable effects dynamically::
cursor.disableEffect(); // Turn off particle effects
cursor.enableEffect();  // Turn on particle effects

// Disable or enable cursor
cursor.disable();
cursor.enable();
